# Olly Auto-Response UX Redesign

This repository contains a frontend prototype built for the Reelo Product Intern assignment. It reimagines the "Setup Auto-Response" flow to balance AI automation speed with precise user control.

## 🔗 Key Links
* **Live Prototype:** [Working Prototype](https://olly-prototype-sage.vercel.app/)
* **Product Brief:** [Notion](https://app.notion.com/p/Product-Brief-Olly-Auto-Response-UX-Redesign-379b4ef90a98808aa84cdcb60331f10b?source=copy_link)
* **Video Walkthrough:** [Watch the 3-minute demo here](https://www.loom.com/share/fe2abe2423f046ebbcc858752bd53019) 

## 💡 The Solution
1. **Confidence-Based Batching:** A "Fast Track" UI to bulk-approve high-confidence AI responses (e.g., standard 5-star reviews) in a single click.
2. **Contextual Inline Editing:** A Grammarly-style interaction for complex reviews, allowing users to seamlessly fix or regenerate specific sentences without losing the rest of the AI's draft.

## 💻 Tech Stack
* React.js (Vite)
* Tailwind CSS
